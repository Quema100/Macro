### 추가해야할 모듈
1.`pip install tkinter`

2.`pip install pyautogui`

3.`pip install keybord`

4.`pip install time`

5.`pip install threading`


# 사용방법

1.exe파일로 만든다
- **만드는 방법**
1. cmd에서 `pip install pyinstaller` 를 하여 모듈을 설치한다.
2. cmd에서 ``를 해서 exe 파일로 만든다.
3. 생섬된 파일 중 `` 파일에 들어가서 실행시킨다.
 

2.자체 실행한다
